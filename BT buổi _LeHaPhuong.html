3. a²-b² = (a-b)*(a+b)

4. (a+b)³ = a³ + 3a²b + 3ab² + b³

5. (a-b)³ = a³ - 3a²b + 3ab² - b³

6. a³ + b³ = (a+b)*(a² -ab + b²)

7. a³ - b³ = (a-b)*(a² +ab + b²)
